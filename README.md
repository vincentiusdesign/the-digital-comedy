# The Digital Comedy
A single-page interactive website structured after Dante Alighieri's epic poem The Divine Comedy, examining the history of the web through three moral lenses.

Hell asks what was done to us. Purgatory asks what was possible. Heaven asks what was aspired to.

## Structure
The project follows Dante's architecture: nine circles of Hell, seven terraces of Purgatory, nine spheres of Heaven, with threshold moments connecting each realm. You descend through Hell by scrolling down, pass through an inversion at the bottom, and ascend through Purgatory and Heaven by scrolling up — arriving at the Empyrean at the top of the page, which is where you began, transformed beyond recognition.

Each realm operates in a categorically different content mode. Hell makes the user perform the victim role — design weaponized against the person experiencing it. Purgatory documents each era of web history using that era's own formal language as the medium. Heaven embodies each vision of what the web could have been at its most idealistic, becoming progressively more sincere as you ascend.

## Concept
Hell is organized by sin, not era — bad design is timeless, and a pop-up from 2003 and a cookie consent dark pattern from 2022 share the same circle. Purgatory is organized by era, moving from table-based layouts through the CSS wars, Flash, Web 2.0, responsive design, and the attention economy. Heaven is organized by vision — the genuine idealism of each period at its peak, honored without qualification, particularly for the visions that were never realized.

The project is built on a single principle: form and content are the same argument made twice. Every design decision is an argument about the web, not a decoration.
